# 403 Bypass

## 403Bypasser

### Installation

```bash
git clone https://github.com/yunemse48/403bypasser.git
cd 403bypasser
sudo pip3 install -r requirements.txt
chmod +x 403bypasser.py
```

### Usage

```bash
python3 403bypasser.py -u https://target.com -d /admin
```
